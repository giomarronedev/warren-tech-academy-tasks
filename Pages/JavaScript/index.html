Exercícios de JavaScript

Durante o módulo de JavaScript, começamos aprendendo lógica de programação, onde aprendi sobre como transformar em código algo que quero que o computador faça. Logo abaixo você pode conferir alguns exercícios de lógica de programação que fiz. Agora, saiba que essa página será atualizada conforme mais exercícios de javascript forem feitos!

Trocando posições de um vetor/array

Um algoritmo que troca a posição de itens de dentro de um vetor.

Invertendo uma array

Esse algoritmo lê uma série de itens informados pelo usuário, os armazena em um vetor e inverte esse vetor. Isso foi feito de forma manual no código, ou seja, sem utilizar atalhos como "array.reverse()".

Retorna os itens de um vetor no console

Um algoritmo que lê 5 números informados pelo usuário e retorna essa lista de itens no console.

Divisores de um número

Um algoritmo que lê um número informado pelo usuário e retorna todos os divisores dele.

Soma de divisores

Um algoritmo que lê um número informado pelo usuário e retorna a soma de todos os divisores dele.

Quantidade de divisores

Um algoritmo que lê um número informado pelo usuário e retorna a quantidade de divisores que ele possui.

Filtro de números positivos

Esse algoritmo lê uma quantidade de números que o usuário quer filtrar e então, o usuário informa a quantidade de números que ele informou. Números positivos e negativos podem ser informados, mas apenas os positivos aparecerão no console.

Contador

Algoritmo que lê um número informado pelo usuário e conta de 1 até o número informado.

Contador em condição

Algoritmo que lê números informados pelo usuário e conta quantos deles são números entre 10 e 20.

Categorização de números

Esse algoritmo lê 5 números informados pelo usuário e diz quantos deles se encaixam nas categorias: 0-25, 26-50, 51-75 e 76-100.

Contador com while

Esse algoritmo adiciona 1 a uma variável enquanto ela for menor que 20.

Números pares

Esse algoritmo retorna no console todos os números pares entre 1 e 50.

Múltiplos de 11

Esse algoritmo retorna no console todos os múltiplos de 11 até o número 200.

Filtro de números pares e positivos

Algoritmo que lê números que o usuário informa mas retorna no console apenas números que sejam pares e positivos.

Filtro de números pares

Algoritmo que lê 5 números que o usuário informa mas retorna no console apenas números que sejam pares.

Ordem crescente

Esse algoritmo lê 2 números que o usuário informa mas retorna no console os mesmos números, mas organizados em ordem crescente.

Retorna número maior

Esse algoritmo lê 2 números que o usuário informa mas retorna no console apenas o maior deles.

Par ou ímpar

Algoritmo que lê um número que o usuário informa e, dependendo do número, retorna no console "0" ou "1", onde "0" significa que o número informado é par e "1" significa que ele é ímpar.

Positivo ou negativo?

Algoritmo que lê um número que o usuário informa e, dependendo do número, retorna no console "0", "1" ou "-1", onde "0" significa que o número informado é 0, "1" significa que ele é positivo e "-1" significa que ele é negativo.

Formatador de segundos

Esse algoritmo lê um número que o usuário informa, converte esses segundos em horas, minutos e segundos e retorna o resultado.

Notas de dinheiro

Algoritmo que lê um número que o usuário informa, e retorna no console quantas notas de 100, 50, 20, 10, 5, 2 e 1 são necessárias para transformar esse número em dinheiro físico.
